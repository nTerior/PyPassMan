# pypassman
An encrypted and secure password manager

## Installation
### Using pip
```bash
sudo pip3 install PyPassMan
pypassman --help
```

### Using Github 
```bash
git clone https://github.com/cephox/pypassman
cd PyPassMan/
sudo pip3 install -r requirements.txt

# Either
python3 pypassman.py --help
# or
./pypassman.py --help
```
